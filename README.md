# 在线考试管理系统 java411

## 项目概述

在线考试管理系统是基于SpringBoot的前后端分离的系统。它集成了考生注册、在线考试、题库管理等多种功能，适用于不同角色的用户需求。

## 技术栈

- 后端框架：SpringBoot、MyBatis-Plus
- 前端框架：Vue、ElementUI、Axios、Node.js
- 数据库：MySQL

## 功能模块

### 系统角色

- 学生
- 老师
- 管理员

### 功能介绍

#### 学生

- 考生注册
- 考生登录
- 在线考试
- 我的成绩
- 我的题库
- 修改个人资料

#### 老师

- 系统登录
- 题库管理
- 试题管理
- 考试管理
- 阅卷管理
- 考试统计
- 修改个人资料

#### 管理员

- 系统介绍
- 题库管理
- 试题管理
- 考试管理
- 阅卷管理
- 考试统计
- 用户管理
- 角色管理

## 运行环境

- 后端：Java 1.8+
- 前端：Node.js环境
- 数据库：MySQL 5.7+

## 部署步骤

1. 准备好MySQL数据库环境
2. 部署后端服务（SpringBoot应用）
3. 部署前端服务（Vue应用）
4. 根据配置文件配置前后端连接

- 基于SpringBoot的微服务架构，易于维护和扩展
- 前后端分离，利于分工合作
- 集成多种用户角色，满足不同用户需求
